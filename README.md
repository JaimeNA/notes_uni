# Notas facu

Estas notas se pueden leer simplemente con un interprete de MarkDown, tienen ecuaciones 
matematicas hechas con Latex embebido en el MarkDown con `$` o `$$`(Para bloques).

Ademas, si se compilan a un archivo pdf(se puede hacer individualmente por materia) se van a 
poder visualizar las imagenes adicionales.

## Como compilar

Se necesita tener instaladas las siguientes dependencias:

- `pandoc`
- `texlive`

Luego, simplemente ir al directorio de la materia deseada y correr el script de bash:

``` bash
./compile_notes.sh
```
